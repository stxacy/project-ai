## Skill Learning

Whenever the user asks to "learn a skill", "save a skill", or "add this as a skill", invoke **Daryl - Project Manager** to:

1. Inspect the skill content and determine the correct `_skills/` subfolder:
   - `css/` — stylesheets, design tokens, utility classes, resets, component styles
   - `js/` — self-contained scripts, helpers, utilities
   - `templates/` — HTML starter structures, page scaffolds
   - `icons/` — SVG icons, sprites, graphic assets
2. Save the skill file into the correct subfolder with a descriptive filename
3. Confirm back to the user where it was placed and why
