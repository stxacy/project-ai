# Team Dashboard — Project Notes

- **Project:** Team Dashboard
- **Goal:** A visual single-page dashboard showing each agent (Daryl, Steven, Cody, Debbie, Pat), their role, tasks completed today, and tokens spent. Uses demo/mock data. Built to test all agents working together.
- **Stack:** Single HTML file, vanilla CSS, vanilla JS
- **Design direction:** Clean, dark dashboard aesthetic. Card per agent. Show task list and token count per card.
