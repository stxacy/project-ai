# Shared Skills

This folder is a central library of reusable skill files, templates, and snippets that can be referenced by any project in Project Ai.

## How to use
- Add a new file here for any pattern, component, or technique you want to reuse across projects.
- Name files clearly using lowercase-kebab-case (e.g., `css-reset.css`, `responsive-nav.html`, `dark-theme-tokens.css`).
- Reference them from a project's notes.md or copy them directly into a project as needed.

## Suggested categories
- `css/`       — reusable CSS snippets, design tokens, utility classes
- `js/`        — reusable JavaScript modules or helpers
- `templates/` — starter HTML templates
- `icons/`     — shared icon files or SVG sprites
